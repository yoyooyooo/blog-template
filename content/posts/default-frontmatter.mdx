tags: [选填]
pathname: [选填] 英文，可以空格，会被转化为小写，-间隔，/ 会被保留，用于 blog 页面内的二级 demo 页
hidden: [选填] 默认值 false，隐藏页面，构建都不进行
folder: [选填]
isPost: [选填] 默认值 true, 是否是 blog，是的话会显示在分页列表和相关文章里
layout: [选填] 默认值 true, 是否显示 Layout(src/components/Layout.tsx)，主要用在 demo 页面
pure: [选填] 默认值 false, 干干净净什么默认 layout 都不加
toc: [选填] 默认值 true
